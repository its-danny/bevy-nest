## Compatibility

| `bevy` | `bevy-nest` |
| :----: | :---------: |
|  0.9   |     0.1     |

## Installation

Add `bevy-nest` to your `Cargo.toml`:

```toml
bevy-nest = "0.1"
```

## Usage

- Dive right into the [docs](https://docs.rs/crate/bevy-nest).
- See the [`examples/`](https://github.com/its-danny/bevy-nest/blob/main/examples) directory for a simple chat server.
